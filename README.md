# AI 小说提示词词条库

这是一个可直接上传到 GitHub Pages 的静态词条收藏网站。

## 功能

- 分类浏览词条组
- 点击词条加入输出区
- 全文搜索词条 / 分类
- 自定义新增词条组
- 可选择任意词条组添加词条
- 每个词条组旁设有「编辑」按钮
- 编辑窗口可修改词条组名称、分类和全部词条内容
- 本地修改会保存到浏览器 LocalStorage
- 支持浅色 / 深色模式

## 上传 GitHub Pages

将本文件夹内的以下内容上传到 repository 根目录：

```text
index.html
README.md
css/
data/
js/
```

然后到 GitHub repository 的 Settings → Pages，选择 Deploy from a branch，Branch 选择 main，Folder 选择 /root。

更新后如仍看到旧界面，请在浏览器按 Ctrl + F5 强制刷新。
