关于

这是一个面向 AI 小说写作的静态词条收藏网站。点击左侧词条后，会直接加入上方选择区与右侧输出框。

自定义词条组会保存到当前浏览器 LocalStorage，不会上传到服务器。上传到 GitHub Pages 后仍可正常使用。

Edit Vocabulary Group

编辑词条组

可修改词条组名称、分类和全部词条内容。词条建议一行一个；也支持用中文逗号、英文逗号或分号分隔。
